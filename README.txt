# STC_test_task

В рамках тестового задания выполнены следующие задачи:
2.1	Реализовать графическое приложение, редактор базы данных населенных пунктов;
2.2	Реализовать многопоточное графическое приложение, отображающее процесс выполнения и завершения выполнения ресурсоемких расчетных процедур в табличном виде;

Исходные коды проектов расположены в папках EditableTableTask1 (для задания 2.1) и MultiThreadTableTask2 (для задания 2.2).

Собранные приложения, скриншоты и установщики хранятся в папках Releases, Screenshots и Installers (Task1 и Task2 соответственно).

Проекты созданы и протестированы в связке Win10 + VS2019 + QT5.15.2 + MSVC v141 (2017)
